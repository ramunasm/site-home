    Site home

    Site Home is an Opera port of a Safari extension that adds a toolbar button designed to take you to the current website’s start (aka. home) page. Original Safari extension by Deep Informational Technologies.

    Download the Site Home extension from Opera’s extension catalog.

    The extension will override the Open History Panel keyboard shortcut (Windows only), Control–h, for it’s own purposes. Supported on all platforms.
